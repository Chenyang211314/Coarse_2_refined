Method schematic

Coarse-to-refined inference flow

This section is intentionally diagram-based rather than video-based, so it can explain the trajectory without requiring an additional process video.

Stage 1

Coarse denoising

Use the simplified prompt at high noise to form subject, layout, and global motion.

Transition

CRTG

Use conditional prediction difference to shift the intermediate latent.

Stage 2

Refined denoising

Use the input prompt at low noise to refine events, attributes, and details.

dcrtg = dc + λ(dp − dc)

Semantic-gap scheduling

The transition point is determined by the gap between the coarse prompt and the input prompt.

Sampling-budget reallocation

Early steps are biased toward high-noise structure formation; later steps are biased toward low-noise detail refinement.

Prompt roles

The coarse prompt stabilizes the main configuration, while the input prompt completes fine-grained constraints.
